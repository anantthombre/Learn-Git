Display Properties in CSS

  • None

    • This property removes the element from the HTML

  • Block

    • This property turns the element into the block element which occupies whole width of the screen for that element

  • inline

    • This property turns the element into the inline element which let other inline elements take the remaining width

  • inline-block

    • This property turns the element into the inline+block element which let other element take free width in the inline and also changes the size of element as its inline + block

    Note: You cannot change the size of inline element
